Exercice 1.6

Écrireun algorithme permettant d’échanger les valeurs de deux variables A et B, et ce quel que soit leur contenu préalable.

REPONSE

Var {A, B, C} : reels;
DEBUT
ecrire ("SAISIR LA VALEUR DE A") ;
lire (A) ;
ecrire ("SAISIR LA VALEUR DE B") ;
lire (B) ;
ecrire ("SAISIR LA VALEUR DE C") ;
lire (C) ;
C -> A;
A -> B;
B -> C;
ecrire (A, B);
FIN
